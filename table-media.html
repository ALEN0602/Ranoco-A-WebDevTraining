MEDIA CSS



Images

Images can be styled in different ways using Materialize

Responsive Images

To make images resize responsively to page width, you can add the class responsive-img to your image tag. It will now have a max-width: 100% and height:auto.

Language-MarkUp

<img class="responsive-img" src="img/image(1).jpg">

Circular Images

To make images appear circular, simply add class="circle" to them

This is a square image. Add the "circle" class to it to make it appear circular.
Language-MarkUp

<div class="col s12 m8">
 <div class="card-panel grey lighten-5 z-depth-2" style="border-radius: 20px; padding-left: 24px;">
  <div class="row valign-wrapper" style="margin: 0%;">
   <div class="col s3">
    <img src="image(4).jpg" alt="" class="responsive-img circle">
   </div>
   <div class="col s10">
    <span class="black-text">
     This is a square image. Add the "circle" class to it to make it appear circular.
    </span>
   </div>
  </div>
 </div>
</div>

Videos

We provide a container for Embedded Videos that resizes them responsively.

Responsive Embeds

To make your embeds responsive, merely wrap them with a containing div which has the class video-container.

Language-MarkUp

<div class="video-container">
  <iframe width="560" height="315" src="https://www.youtube.com/embed/eoTpdTU8nTA?controls=0" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
</div>

Responsive Videos

To make your HTML5 Videos responsive just add the class responsive-video to the video tag.

Language-MarkUp

<video class="responsive-video" controls>
  <source src="nightsky.mp4" type="video/mp4" width="560" height="315">
</video>
